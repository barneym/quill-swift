# Extended Checkbox Examples

QuillSwift supports extended checkbox syntax beyond the standard `[ ]` and `[x]`.

## Standard Checkboxes

- [ ] Unchecked task
- [x] Completed task
- [X] Also completed (uppercase X)

## Extended States

### Progress States
- [/] In progress - started but not complete
- [-] Blocked or on hold

### Priority/Importance
- [!] Important/high priority
- [?] Question/needs review
- [*] Star/favorite

### Status Indicators
- [>] Forwarded/delegated
- [<] Scheduled/deferred
- [~] Cancelled/deprecated

- [ ] Main task

  - [/] Subtask 2 (in progress)
